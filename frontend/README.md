# InterviewForge Frontend

Independent Next.js App Router frontend for InterviewForge. It renders product UI and calls the Fastify backend for business operations; do not add Next.js Route Handlers or Server Actions as business API substitutes.

```powershell
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```

See the repository root `README.md`, `PRODUCT.md`, and `DESIGN.md` before implementing features.
